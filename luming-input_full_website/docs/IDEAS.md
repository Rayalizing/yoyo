# 麓鸣输入法网站设计方案

「武侠水墨」风格

### 设计
融合传统中国水墨画与现代极简主义，呼应麓鸣的武侠命名体系（北冥、无相、折梅）。

### 核心原则
1. **意境留白**：大量使用留白，让内容如山水画般呼吸
2. **笔墨韵味**：使用水墨渐变、毛笔笔触元素
3. **动静结合**：静态的禅意与动态的剑气并存
4. **古今融合**：传统美学与现代交互的平衡

### 色彩哲学
- 主色：墨黑 (#1a1a1a) 与宣纸白 (#f8f6f0)
- 点缀：朱砂红 (#c94a4a) 用于强调
- 辅助：淡墨灰 (#6b6b6b) 用于次要文字
- 情感意图：沉稳、雅致、有文化底蕴

### 布局范式
- 非对称布局，模仿传统卷轴展开
- 垂直滚动为主，如同展开长卷
- 内容区域如印章般有明确边界

### 标志性元素
1. 水墨晕染背景动效
2. 毛笔书法风格的标题
3. 印章式的按钮和标签

### 交互哲学
- 悬停时如墨滴入水般扩散
- 页面切换如翻书卷
- 滚动时有轻微的视差效果

### 动画指南
- 入场动画：元素如墨迹般渐显
- 过渡动画：柔和的淡入淡出，300-500ms
- 微交互：轻盈的弹性效果

### 字体系统
- 标题：霞鹜文楷 / 思源宋体（衬线，有书卷气）
- 正文：思源黑体 / Noto Sans SC（清晰易读）
- 代码：JetBrains Mono

理由：
1. 与麓鸣的武侠命名体系（北冥神功、小无相功、折梅）高度契合
2. 水墨风格在输入法网站中独树一帜，具有强烈辨识度
3. 能够体现中文输入法的文化属性
4. 留白美学与麓鸣「极简规则」的理念相呼应
